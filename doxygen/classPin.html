Detailed Description

A class for controlling GPIO pins on Raspberry Pi.

This class provides a simple interface for controlling GPIO pins, allowing you to set pins high/low and read their current state.

int Pin::read ( )

Read the current state of the pin.

Reads the digital state of a GPIO pin configured as INPUT.

Returns
int Returns 1 if the pin is HIGH (3.3V), 0 if the pin is LOW (0V), -1 on error
Note
Works on pins configured as INPUT, INPUT_PULLUP, or INPUT_PULLDOWN.
For floating inputs without pull resistors, readings may be unstable.
Pin button(18, PinMode::INPUT_PULLUP);
int state = button.read();
if (state == 0) {
// Button is pressed (connected to GND)
}

bool Pin::write ( bool  value)

Write a value to the pin.

Sets the output state of a GPIO pin configured as OUTPUT.

Parameters
valuetrue to set the pin HIGH (3.3V), false to set it LOW (0V)
Returns
bool Returns true on success, false on failure
Note
Only works on pins configured as OUTPUT. Input pins will return false.
Warning
Ensure external circuitry can handle 3.3V logic levels.
Pin led(17, PinDirection::OUTPUT);
led.write(true); // Turn LED on
led.write(false); // Turn LED off
