Detailed Description

Arduino-inspired SPI communication library using Linux spidev.

Author
Barbatos6669
Date
2025

Provides SPI master communication compatible with Arduino's SPI library API. Uses Linux kernel spidev interface for hardware SPI access on Raspberry Pi.

Features:

  • Arduino SPI API compatibility
  • Hardware SPI (up to 125 MHz on Pi 4/5)
  • Configurable mode (0-3), bit order, clock speed
  • Full-duplex communication
  • Thread-safe operations
