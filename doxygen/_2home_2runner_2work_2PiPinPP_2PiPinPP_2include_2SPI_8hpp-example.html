Initialize SPI interfaceOpens the default SPI device (SPI0.0 on Raspberry Pi). Configures default settings: MODE0, MSBFIRST, 4 MHz.

Returns
true if successful, false on error
Note
Raspberry Pi SPI pins:
  • MOSI: GPIO 10 (Pin 19)
  • MISO: GPIO 9 (Pin 21)
  • SCLK: GPIO 11 (Pin 23)
  • CE0: GPIO 8 (Pin 24)
  • CE1: GPIO 7 (Pin 26)

SPI.begin(); // Initialize with defaults
