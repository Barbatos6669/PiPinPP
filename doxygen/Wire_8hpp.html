Detailed Description

Arduino-inspired I2C communication library using Linux i2c-dev.

Author
Barbatos6669
Date
2025

Provides Wire library API compatible with Arduino for I2C master communication. Uses Linux kernel i2c-dev interface for hardware I2C access.

Features:

  • Arduino Wire API compatibility
  • I2C master mode
  • 7-bit and 10-bit addressing support
  • Configurable clock speed
  • Read/write operations
  • Register-based access helpers
