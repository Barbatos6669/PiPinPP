◆ pulseInLong()

unsigned long pulseInLong ( int  pin,
bool  state,
unsigned long  timeout = 1000000 
)
inline

Measure the length of a pulse on a pin with extended range (Arduino-inspired)

Similar to pulseIn() but returns unsigned long for measuring longer pulses. Use this for pulses longer than ~71 minutes.

Parameters
pinGPIO pin number to read
statePulse state to measure (HIGH or LOW)
timeoutMaximum time to wait in microseconds (default: 1000000 = 1 second)
Returns
unsigned long Pulse length in microseconds, or 0 if timeout
Note
Identical to pulseIn() on Raspberry Pi (both return unsigned long)
Provided for Arduino code compatibility
