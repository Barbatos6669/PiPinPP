### Arduino Math Function: `sq()`

Square a value efficiently with Arduino-compatible syntax:

```cpp
#include <ArduinoCompat.hpp>

int x = 5;
int result = sq(x);  // Returns 25

// Works with any numeric type
float voltage = 3.3;
float power = sq(voltage);  // Returns 10.89

// Useful for calculations
int distance = sq(width) + sq(height);  // Pythagorean theorem
```

**Features:**
- Generic template supporting all numeric types (int, float, double, etc.)
- Inline for zero-overhead abstraction
- Arduino-compatible syntax
- No conflicts with standard library

**Note:** For other math functions, use standard library:
- `std::sqrt()` - Square root
- `std::pow()` - Power function
- `std::max()` - Maximum of two values
- `std::min()` - Minimum of two values

This design avoids naming conflicts with `using namespace std;` and leverages well-optimized standard implementations.
