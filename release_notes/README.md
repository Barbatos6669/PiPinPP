# Release Notes

This directory contains detailed release notes for each version of PiPinPP.

## Format

Each release has its own markdown file: `vX.X.X.md`

These notes provide:
- Feature highlights and examples
- Migration guides
- Installation instructions
- Testing results
- Bug fixes and improvements

## Usage

When creating a GitHub Release, copy the contents of the corresponding file to the release description.
