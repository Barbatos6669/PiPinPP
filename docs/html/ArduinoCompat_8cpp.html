◆ digitalToggle()

void digitalToggle ( int  pin)

Toggle the state of an OUTPUT pin (HIGH↔LOW)

Efficiently toggles pin state without reading current value. Much faster than digitalRead() + digitalWrite().

Thread-safe: Multiple threads can call this function concurrently.

Parameters
pinGPIO pin number
Exceptions
PinErrorif pin not initialized or not OUTPUT
Note
Example:
pinMode(17, OUTPUT);
digitalWrite(17, HIGH);
digitalToggle(17); // Now LOW
digitalToggle(17); // Now HIGH
